# Fragile States Index Analysis
### ITCS 6162 — Knowledge Discovery in Databases

## Project Overview
This project analyzes country-level state fragility using the
Fragile States Index (FSI) dataset across 2019–2021. The goal is to
understand what factors drive state fragility and — more importantly —
what actionable changes can help countries become more stable.

We combine 3 years of FSI data (178–179 countries per year) with
4 additional features from the World Bank API, then apply machine
learning classification and action rule mining to extract insights.

---

## Problem Statement
State fragility is a complex phenomenon driven by security breakdowns,
economic collapse, political illegitimacy, and social pressures. The
FSI scores 178+ countries annually across 12 sub-indicators. A lower
score means greater stability — a higher score means greater fragility.

The key questions we answer:
- What combination of factors best predicts a country's fragility level?
- What specific changes in those factors could move a country to a
  less fragile category?

## Dataset
### FSI Data (Primary)
- **Source**: Fund for Peace — https://fragilestatesindex.org/excel/
- **Years**: 2019, 2020, 2021
- **Countries**: 179 unique countries
- **Records**: 535 total (after merging 3 years)
- **Decision Feature**: FSI Total Score → discretized into 9 categories
  following the Wikipedia FSI classification:
  - Very High Alert (90–120)
  - High Alert     (80–90)
  - Alert          (70–80)
  - High Warning   (60–70)
  - Warning        (50–60)
  - Stable         (40–50)
  - More Stable    (30–40)
  - Sustainable    (20–30)
  - Very Sustainable (below 20)

### Original FSI Sub-Indicators (12 features)
| Code | Name |
|------|------|
| C1 | Security Apparatus |
| C2 | Factionalized Elites |
| C3 | Group Grievance |
| E1 | Economy |
| E2 | Economic Inequality |
| E3 | Human Flight and Brain Drain |
| P1 | State Legitimacy |
| P2 | Public Services |
| P3 | Human Rights |
| S1 | Demographic Pressures |
| S2 | Refugees and IDPs |
| X1 | External Intervention |

### 4 New Features Added (from World Bank API)
| Feature | Indicator Code | Correlation with FSI | Reason |
|---------|---------------|----------------------|--------|
| GDP per Capita | NY.GDP.PCAP.CD | -0.779 | Economic poverty drives fragility |
| Life Expectancy | SP.DYN.LE00.IN | -0.793 | Reflects healthcare & conflict impact |
| Internet Access | IT.NET.USER.ZS | -0.722 | Proxy for development & openness |
| Infant Mortality | SP.DYN.IMRT.IN | +0.723 | Reflects basic services & healthcare |

All 4 features show strong correlation (|r| > 0.72) with FSI score,
confirming they are meaningful additions to the dataset.

## Methodology

### 1. Data Preprocessing
- Merged 3 annual FSI Excel files into one dataset
- Removed "Change from Previous Year" column (only in 2019 & 2020)
- Standardized country names to World Bank format
- Fetched 4 new features via World Bank REST API
- Applied median imputation for missing values
- Discretized FSI score into 9 categories

### 2. Classification
Three classifiers were trained and compared with and without
the 4 new features:

| Classifier | Accuracy (Original) | Accuracy (+4 Features) | Change |
|------------|--------------------|-----------------------|--------|
| Decision Tree | 65.4% | 62.6% | -2.8% |
| Random Forest | 75.7% | **79.4%** | +3.7% |
| Naive Bayes | 71.9% | 68.2% | -3.7% |

Random Forest achieved the best performance and improved with
the addition of the 4 new features, confirming their value.

### 3. Action Rule Mining
Action rules were extracted using LISp-Miner (Ac4ft-Miner module).
Action rules identify what changes in flexible attributes are needed
to move a country from a more fragile to a less fragile FSI category.

**Stable attributes** (cannot change): Country, Year
**Flexible attributes** (can change): All 12 FSI sub-indicators +
4 new features
**Decision**: FSI_Category (target — we want to move toward lower fragility)

## Key Findings
- **Random Forest** is the best classifier (79.4% accuracy with new features)
- **Life Expectancy** and **GDP per Capita** are the strongest new predictors
- All 4 new features show correlation above 0.72 with FSI score
- Action rules suggest that improving **public services**, **economy**,
  and **GDP per capita** are the most impactful levers for reducing fragility
