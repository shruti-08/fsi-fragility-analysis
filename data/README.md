# Data

## Raw FSI Data
The raw FSI Excel files are NOT included in this repository due to size.

Download them manually from:
https://fragilestatesindex.org/excel/

Save them as:
- data/raw/fsi-2019.xlsx
- data/raw/fsi-2020.xlsx
- data/raw/fsi-2021.xlsx

## World Bank Data
All World Bank indicators are fetched automatically via the
World Bank API inside the notebook. No manual download needed.

## Processed Data
Processed files are generated by running the notebook:
- data/processed/fsi_merged_complete.csv  — merged dataset with all features
- data/processed/fsi_lisp_miner.csv       — discretized dataset for LISp-Miner
